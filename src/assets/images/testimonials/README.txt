Папка для фотографий к отзывам

Рекомендации по загрузке:
- Размер изображений: 100x100 пикселей (аватары), 800x600 пикселей (фото к отзывам)
- Формат: JPEG (.jpg) или PNG (.png) для аватаров
- Оптимизируйте изображения перед загрузкой
- Рекомендуемый размер файла: не более 50 КБ для аватаров, 300 КБ для фото

Именование файлов:
- Используйте только латинские буквы, цифры и дефисы
- Для аватаров: avatar-1.jpg, avatar-2.png
- Для фото к отзывам: review-photo-1.jpg, review-photo-2.jpg

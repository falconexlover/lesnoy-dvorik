Папка для изображений номеров гостиницы

Структура папок:
- econom/ - Номера категории "Эконом"
- standard/ - Номера категории "Стандарт"
- family/ - Семейные номера
- comfort/ - Номера категории "Комфорт"
- lux/ - Номера категории "Люкс"

Рекомендации по загрузке:
- Размер изображений: 1200x800 пикселей
- Формат: JPEG (.jpg)
- Оптимизируйте изображения перед загрузкой
- Рекомендуемый размер файла: не более 500 КБ

Именование файлов:
- Используйте только латинские буквы, цифры и дефисы
- Пример: econom-room-1.jpg, standard-room-bathroom-2.jpg

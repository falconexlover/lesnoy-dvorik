Папка для изображений галереи

Структура папок:
- all/ - Общие фотографии для галереи

Рекомендации по загрузке:
- Размер изображений: 1200x800 пикселей для оригиналов, 600x400 для миниатюр
- Формат: JPEG (.jpg)
- Оптимизируйте изображения перед загрузкой
- Рекомендуемый размер файла: не более 500 КБ для оригиналов, 200 КБ для миниатюр

Именование файлов:
- Используйте только латинские буквы, цифры и дефисы
- Для миниатюр добавляйте суффикс "-thumb"
- Пример: gallery-room-1.jpg, gallery-room-1-thumb.jpg

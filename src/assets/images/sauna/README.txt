Папка для изображений сауны и бани

Структура папок:
- russian_bath/ - Русская баня
- finnish_sauna/ - Финская сауна
- pool/ - Бассейн
- relax_room/ - Комната отдыха

Рекомендации по загрузке:
- Размер изображений: 1200x800 пикселей
- Формат: JPEG (.jpg)
- Оптимизируйте изображения перед загрузкой
- Рекомендуемый размер файла: не более 500 КБ

Именование файлов:
- Используйте только латинские буквы, цифры и дефисы
- Пример: russian-bath-interior-1.jpg, pool-area-2.jpg

Папка для изображений конференц-зала

Структура папок:
- hall/ - Общие виды зала
- equipment/ - Оборудование
- events/ - Проведенные мероприятия

Рекомендации по загрузке:
- Размер изображений: 1200x800 пикселей
- Формат: JPEG (.jpg)
- Оптимизируйте изображения перед загрузкой
- Рекомендуемый размер файла: не более 500 КБ

Именование файлов:
- Используйте только латинские буквы, цифры и дефисы
- Пример: conference-hall-main.jpg, conference-equipment-projector.jpg

Папка для изображений ресторана

Структура папок:
- interior/ - Интерьер ресторана
- dishes/ - Блюда
- menu/ - Фотографии меню

Рекомендации по загрузке:
- Размер изображений: 1200x800 пикселей для интерьера, 800x600 для блюд
- Формат: JPEG (.jpg)
- Оптимизируйте изображения перед загрузкой
- Рекомендуемый размер файла: не более 500 КБ

Именование файлов:
- Используйте только латинские буквы, цифры и дефисы
- Пример: restaurant-interior-1.jpg, dish-beef-stroganoff.jpg

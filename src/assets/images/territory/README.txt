Папка для изображений территории гостиницы

Структура папок:
- exterior/ - Внешний вид здания
- garden/ - Сад и ландшафт
- recreation/ - Зоны отдыха

Рекомендации по загрузке:
- Размер изображений: 1200x800 пикселей
- Формат: JPEG (.jpg)
- Оптимизируйте изображения перед загрузкой
- Рекомендуемый размер файла: не более 500 КБ

Именование файлов:
- Используйте только латинские буквы, цифры и дефисы
- Пример: hotel-exterior-front.jpg, garden-flowers-1.jpg

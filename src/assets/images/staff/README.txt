Папка для фотографий персонала

Рекомендации по загрузке:
- Размер изображений: 600x600 пикселей (квадратные)
- Формат: JPEG (.jpg)
- Оптимизируйте изображения перед загрузкой
- Рекомендуемый размер файла: не более 200 КБ

Именование файлов:
- Используйте только латинские буквы, цифры и дефисы
- Пример: staff-manager.jpg, staff-receptionist.jpg

# Необходимые изображения для сайта "Лесной дворик"

Для корректной работы сайта требуются следующие изображения. Удостоверьтесь, что они присутствуют в соответствующих директориях:

## Основные изображения
- logo.png - логотип сайта (рекомендуемый размер: 200x60px, формат PNG с прозрачностью)
- logo-light.png - светлая версия логотипа для футера
- hotel-main.jpg - главное изображение отеля для верхнего слайдера (рекомендуемый размер: 1920x1080px)
- hotel-main-medium.jpg - среднее изображение гостиницы (для планшетов)
- hotel-main-small.jpg - маленькое изображение гостиницы (для мобильных)
- hotel-about.jpg - изображение для секции "О нас" (рекомендуемый размер: 800x600px)
- room-standard.jpg - изображение стандартного номера (рекомендуемый размер: 800x600px)
- room-comfort.jpg - изображение комфорт-номера (рекомендуемый размер: 800x600px)
- room-luxury.jpg - изображение люкс-номера (рекомендуемый размер: 800x600px)
- room-business.jpg - изображение бизнес-номера (рекомендуемый размер: 800x600px)
- og-image.jpg - изображение для Open Graph (соцсети)

## Favicon изображения
Эти файлы должны быть размещены в директории `/assets/favicon/`:
- favicon.ico - основная иконка сайта (размеры 16x16, 32x32 и 48x48px)
- favicon-16x16.png - иконка 16x16px для браузеров
- favicon-32x32.png - иконка 32x32px для браузеров
- apple-touch-icon.png - иконка для устройств Apple (размер 180x180px)
- safari-pinned-tab.svg - векторная иконка для вкладок Safari
- android-chrome-192x192.png - иконка для Android (размер 192x192px)
- android-chrome-512x512.png - иконка для Android (размер 512x512px)
- mstile-150x150.png - иконка для Windows (150x150)

## Рекомендуемые размеры
- Логотип: 250x70 px
- Баннеры: 1920x600 px
- Изображения номеров: 800x500 px
- Изображения для галереи: 600x400 px
- OG изображение: 1200x630 px

Примечание: Все изображения должны быть оптимизированы для веба (JPEG/WebP для фотографий, PNG для логотипов с прозрачностью).

Если у вас нет подходящих изображений, вы можете:

1. Использовать стоковые фотографии с бесплатных ресурсов, таких как:
   - Unsplash (https://unsplash.com/)
   - Pexels (https://www.pexels.com/)
   - Pixabay (https://pixabay.com/)

2. Создать временные заглушки на сервисах:
   - Placeholder.com (например: https://via.placeholder.com/400x300/5a8f7b/ffffff?text=Лесной+дворик)
   - DummyImage (https://dummyimage.com/)

ВАЖНО: При загрузке изображений убедитесь, что имена файлов совпадают с указанными выше, иначе они не будут отображаться на сайте. Также убедитесь, что все изображения оптимизированы для веб (сжаты) для ускорения загрузки страниц.

Основные заголовки

На нашем сайте используется 6 уровней заголовков, основанных на переменных размеров шрифтов.

Заголовок H1 (--h1-font-size)

Основной заголовок страницы, используется только один раз на странице.

Заголовок H2 (--h2-font-size)

Заголовок секции, разделяет основные смысловые блоки контента.

Заголовок H3 (--h3-font-size)

Подзаголовок секции, используется для групп связанного контента внутри секции.

Заголовок H4 (--h4-font-size)

Заголовок подсекции, используется для более детального разделения контента.

Заголовок H5 (--h5-font-size)

Используется для выделения небольших разделов или групп элементов.

Заголовок H6 (--h6-font-size)

Наименьший заголовок, используется редко, только для особых случаев.

Текстовые элементы

Основные текстовые элементы сайта, используемые для контента.

Вводный абзац (Lead). Используется в начале секции для привлечения внимания и введения в тему. Имеет больший размер шрифта и более свободное расстояние между строками.

Обычный параграф текста. Основной текстовый элемент для содержимого сайта. Используется шрифт Roboto с базовым размером 16px и нормальным межстрочным интервалом.

При использовании нескольких параграфов они разделяются отступом, определенным в переменной --paragraph-spacing. Это обеспечивает согласованные интервалы между блоками текста.

Полужирный текст используется для выделения важных частей в предложении.

Курсив используется для выделения терминов или для создания акцента.

Выделенный текст используется для привлечения внимания к ключевым фразам.

Текст может содержать ссылки на другие страницы, которые выделяются цветом и подчеркиванием при наведении.

Также возможно использование мелкого текста для примечаний или дополнительной информации.

Цитаты оформляются в блоке с отступом и вертикальной линией слева. Это помогает визуально отделить цитируемый текст от основного содержимого.

Если цитата имеет несколько параграфов, они все будут включены в один блок цитаты.

Примеры кода могут быть встроены в текст с помощью элемента <code>, который использует моноширинный шрифт.

// Пример блока кода
function showNotification(message) {
    const notification = document.createElement('div');
    notification.className = 'notification';
    notification.textContent = message;
    document.body.appendChild(notification);
}

Адаптивная типографика

Наша типографика автоматически адаптируется под различные размеры экранов:

  • Десктоп (> 992px): Базовый размер шрифта 16px, заголовок H1 - 36px
  • Планшет (768px - 992px): Базовый размер шрифта 16px, заголовок H1 - 32px
  • Смартфон (576px - 768px): Базовый размер шрифта 15px, заголовок H1 - 28px
  • Малые экраны (< 576px): Базовый размер шрифта 14px, заголовок H1 - 24px

Кроме того, система учитывает пользовательские настройки доступности:

  • Режим с ограниченной анимацией (prefers-reduced-motion)
  • Режим высокой контрастности (prefers-contrast: high)
